# test

This directory contains Python scripts for processing behaviour data for test.

## Scripts
- `getPdfInfo.py`: Extract behaviour notes from PDFs
- `getExcelInfo.py`: Process incident reports from Excel
- `getBe.py`: Generate behaviour analysis
- `upload_to_dashboard.py`: Upload to Firebase
- `update.py`: Sync with Firebase

## Directories
- `downloads/`: Uploaded files are stored here
- `files/`: Additional data files
- `analyzed/`: Processed output files
